📋 Supabase Dashboard Checklist
1.
Add a redirect URL: Authentication → URL Configuration → add https://[your-github-pages-url]/ to Redirect URLs. Without this, password reset links won't work.
2.
Invite a user: Authentication → Users → Invite User → enter their email. They'll receive a setup link. When they click it they land here to set a password.
3.
Disable email confirmation (optional): Authentication → Providers → Email → disable "Confirm email" for internal apps so users don't need to verify first.
4.
Reset a password: Use the "Reset Email" button in the table above. This sends the user a password reset link to their registered email.
